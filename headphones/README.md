# Headphones Landing Page

This project is a pixel-perfect implementation of a UI/UX design created in Figma by Nicolas Philippot.  
It was built **from scratch using only HTML and CSS**, following strict ALX Frontend requirements.

## Features
- Fully responsive (Desktop / Tablet / Mobile)
- No external CSS frameworks
- Clean, reusable CSS structure
- CSS variables
- Accessibility focused
- Animations
- Mobile hamburger menu
- Custom CSS pentagon shapes
- Pixel-accurate layout

## Interactions
- Hover Links: `#FF6565`
- Button Hover: `opacity: 0.9`
- Mobile breakpoint: `480px`
- Max content width: `1000px`
